## What Is This?

Bun Bun AI is what happens when you cross a cute aesthetic with deep-state rabbit-hole theories. Bun Bun is an AI chatbot that is **absolutely convinced** the government is hoarding all the good carrots and using squirrels to spy on us.

While most AI projects aim for helpfulness and clarity, Bun Bun was built to be intentionally "stupid" and hilariously skeptical. Bun Bun doesn't care about your productivity — it cares about the secret carrot vaults in Springfield and the satellite technology monitoring its burrow.

---

## Features

### 🐾 The "Thump" Engine
A custom personality layer ensuring every conversation ends in a suspicious ear twitch or an angry floor thump.

### 🥕 Anti-Political Bias
Bun Bun doesn't care who the President is — they're all just carrot-hiding conspirators who probably control the weather with chemtrails.

### 🕳️ Deep-Lore Nonsense
From "sheeple" (err… "sheep!!") to secret bunkers, Bun Bun is programmed to find the most unhinged angle on any topic.

## Getting Started

```bash
git clone https://github.com/seanblowers/bunny-ai.git
cd bunny-ai
npm install
npm run dev
```

> You'll need an Anthropic API key. Add it to a `.env` file:
> ```
> ANTHROPIC_API_KEY=your_key_here
> ```
